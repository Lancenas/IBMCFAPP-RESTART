# IBMCFAPP-RESTART
IBM CLOUD APPS RESTART  
***
利用Github的Actions 自定义重启 IBM Cloud Fonudray  
---
Fork项目下settings-secret-New secret 分别建立以下四项：  
```
MAIL // IBM Cloud登录邮箱
PWD  // IBM CLOUD登陆密码
CFNAME // 应用的名称
RGN // us-south
```
